Projects

GunPUNK

GunPUNK is a turn-base strategic shoot where your goal as the player is to bounce lasers off the environment to kill enemies. This project was developed using UE4 in about 15 weeks by four other students and I for our major project at AIE. I developed the core game systems in C++, and the AI using behaviour trees. I also created some materials/shaders using the material editor and HLSL.

Medieval Ninja Saloon

Medieval Ninja Saloon was created for a two day gamejam for students at AIE. We given the random theme of medieval ninja saloon, and this is what we came up with. This game was developed in UE4, and I created all of the games systems using blueprints.

Dawn VR

Dawn is a VR horror game for the Occulus Go. Your aim is to defend yourself against ghosts until morning, while conserving you torch's battery. Dawn was developed using UE4 in about 3 weeks for a group assessment for AIE. I programmed the core game systems in C++ and using blueprints for game mechanics.
